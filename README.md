# crud-python
Python Flask Vuejs CRUD (Create, Read, Update and Delete) com SQLTools

Aqui estão algumas etapas para criação um app Python Flask VueJS CRUD (Criar, Ler, Atualizar e Excluir)

- Criação de um banco de dados e tabela MySQL.
- Instalação Flask e VueJS.
- Criação um app Flask.
- Conectar o app Flask ao banco de dados MySQL.
- Criação das rotas CRUD no aplicativo Flask.
- Criação os componentes VueJS.
- Implantar o app.
